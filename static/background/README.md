# 登录页面背景资源

此目录用于存放登录页面的背景视频或图片。

## 支持的文件格式

### 视频背景（推荐，电脑端显示效果最佳）
- **MP4**: `login_bg.mp4` （推荐格式，兼容性最好）
- **WebM**: `login_bg.webm` （备选格式，文件更小）

### 图片背景（备用）
- **JPG/JPEG**: `login_bg.jpg` （推荐，文件较小）
- **PNG**: `login_bg.png` （支持透明，但文件较大）

## 使用方法

### 方式一：使用视频背景（推荐）

1. 将视频文件重命名为 `login_bg.mp4` 或 `login_bg.webm`
2. 放入此目录 (`static/background/`)
3. 刷新登录页面即可看到视频背景

**视频要求：**
- 建议分辨率：1920x1080 或更高
- 建议时长：10-30秒（会自动循环播放）
- 建议格式：MP4 (H.264编码)
- 文件大小：建议控制在 5-10MB 以内

### 方式二：使用图片背景

1. 将图片文件重命名为 `login_bg.jpg` 或 `login_bg.png`
2. 放入此目录 (`static/background/`)
3. 如果没有视频文件，系统会自动使用图片作为背景

**图片要求：**
- 建议分辨率：1920x1080 或更高
- 建议格式：JPG（文件较小）或 PNG（支持透明）
- 文件大小：建议控制在 2-5MB 以内

## 优先级

系统会按以下优先级加载背景：
1. **视频背景**（电脑端优先显示，移动端自动隐藏以节省流量）
2. **图片背景**（如果视频不存在）
3. **SVG线条小狗**（如果视频和图片都不存在，使用默认的线条小狗背景）

## 注意事项

- 视频背景在移动端（≤768px）会自动隐藏，改为显示图片或SVG背景，以节省流量和提升性能
- 背景会自动添加半透明遮罩层，确保登录表单的可读性
- 视频会自动循环播放，静音播放，不会影响用户体验
- 建议使用与主题色（蓝色系）搭配的背景素材

## 推荐资源

您可以使用以下类型的背景：
- 可爱的线条小狗动画视频
- 简洁的几何图形动画
- 柔和的渐变背景视频
- 与编程/Python相关的主题背景
